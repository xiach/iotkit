# iotkit
Full stack of IOT cloud.

nginx: configuration of nginx

service: systemd based service config

smart/admin: admin sites of cloud

smart/backend: core of cloud

smart/webapp: wechat pub sites

iot env:
1) nginx: 1.13
2) uwsgi: 2.0
3) mongodb: 3.6
4) redis: 3.0
5) mariadb: 5.5
6) mosquitto: 1.15
7) python: 3.5
